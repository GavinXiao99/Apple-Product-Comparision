# Apple 产品全球价格对比

这个仓库已经包含可直接发布的静态网站：`index.html`。

正式访问地址启用后会是：

https://gavinxiao99.github.io/News-Tracker/

如果这个地址暂时显示 404，请在 GitHub 仓库中打开：Settings → Pages，然后选择：

- Source: Deploy from a branch
- Branch: `gh-pages` / `/root`

保存后等待 1-2 分钟即可访问。仓库里也已放入 GitHub Actions Pages workflow，后续也可以切换为 GitHub Actions 发布。
